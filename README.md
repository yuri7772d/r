
## install nuxt

```bash

npm create nuxt@latest <name>

```

## install tailwind

```bash

npm create nuxt@latest <name>
# install tailwind เเบบเพิ่ม config ไห้เลย
npx nuxi module add tailwindcss



```
 หรือ

```bash
   npm install -D @nuxtjs/tailwindcss

   # เพิ่ม ลงใน  nuxt.config.ts

   modules: ["@nuxtjs/tailwindcss"]
```
